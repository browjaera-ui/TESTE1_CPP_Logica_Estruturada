# Resolução do Teste I - Laboratório II (C++)

Este repositório contém a resolução do Teste I da cadeira de Laboratório II (C++).

## Descrição dos Exercícios

1. **Pergunta 1**: Programa para armazenar e calcular a média semanal de temperaturas, indicando o dia mais quente.
2. **Pergunta 2**: Sistema de stock utilizando vetores paralelos com alerta de "Stock Crítico" para quantidades inferiores a 5.
3. **Pergunta 3**: Gestão de notas de uma turma de 15 alunos, listando aprovados e calculando a percentagem de aprovação.
4. **Pergunta 4**: Sistema de faturação para 5 itens com aplicação automática de desconto de 10% em compras superiores a 1000 MT.
5. **Pergunta 5**: Ordenação de 8 atletas pelo tempo de corrida (do mais rápido para o mais lento), mantendo a integridade dos dados (nome e tempo).
6. **Pergunta 6**: Sistema de consulta de preços para 20 produtos utilizando vetores paralelos.

## Como Executar

Para compilar qualquer um dos ficheiros, utilize o compilador `g++`:

```bash
g++ perguntaX.cpp -o perguntaX
./perguntaX
```
